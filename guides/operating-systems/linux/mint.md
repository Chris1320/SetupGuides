# Linux Mint

## Installation

1. Download the [Linux Mint ISO](https://linuxmint.com/download.php).[^1]
2. Flash the ISO to your USB and boot to the Live CD.[^2]
3. Install Linux Mint. Follow the instructions.
4. Follow Linux Mint Welcome "First Steps" setup guide.
    - Setup Timeshift.
    - Update drivers via the Driver Manager.
    - Update packages via the Update Manager or by running `sudo apt update && sudo apt upgrade`.
    - Reboot.

## Customization

- Install [zsh](../shells/zsh.md).
- Perform the [neovim](../text-editors/neovim.md) customization guide.
- Follow recommended [Firefox](../browser/firefox-nightly.md) customizations.

[^1]: It is recommended to choose the *Cinnamon* or *Cinnamon EDGE* editions.
[^2]: It is recommended to use [Ventoy](https://www.ventoy.net/en/index.html).
