## Arch Linux

This guide will help you set up Arch Linux.

> **NOTE**: This guide assumes that you are experienced in flashing ISO images.

## Installation

1. Download the ISO image and its signature from [Arch Linux](https://www.archlinux.org/download/).
2. Verify the image. `PS C:\> gpg --keyserver-options auto-key-retrieve --verify archlinux-version-x86_64.iso.sig`
3. Flash the ISO image to your preferred medium. I recommend using a USB flash drive.
4. Boot the Arch Linux ISO image.
5. Install the [Arch Linux bootloader](https://wiki.archlinux.org/index.php/Installation_Guide#Boot_Loader).
